Getting Started

Prerequisites
    Git
    Node.js and npm Node >= 4.x.x, npm >= 2.x.x
    Gulp (npm install --global gulp)
    MongoDB - Keep a running daemon with mongod


Developing
    Run npm install to install server dependencies.
    Run mongod in a separate shell to keep an instance of the MongoDB Daemon running
    Run gulp serve to start the development server. It should automatically open the client in your browser when ready. (Take some time, be patient)
    Build & development
    Run gulp build for building and gulp serve for preview.
